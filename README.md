# Urban Ants
An optimal routes finding algorithm between two points on a city, minimizing both time and cost and considering urban public buses. The algorithm has been built following Ant Colony Optimization (ACO) methodology.

More details about the methodology can be found at:
- [paper work](./docs/hormigas%20urbanas%20-%202018.pdf)
- [joaquin's portfolio](https://joagzb.com)

## technologies and libraries
* python 3.X
* numpy, pandas and Matplotlib
* OpenStreetMaps
